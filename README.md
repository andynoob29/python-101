# Python-101

Collection of Python scripts and web applications.

## Projects

### 1. Python Calculator (calculator.py)
A command-line calculator with the following operations:
- Addition
- Subtraction
- Multiplication
- Division
- Power
- Logarithm

**Usage:**
```bash
python3 calculator.py
```

### 2. Web Calculator (calculator.html)
A beautiful retro-futuristic web calculator with the same operations.

**Usage:**
Simply open `calculator.html` in your web browser, or visit the live demo.
